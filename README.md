# 📊 Loan Default Risk Analysis

## 📌 Overview

This project analyzes loan applicant data to identify key factors influencing loan approval and default risk. The goal is to support data-driven decision-making by understanding customer financial behavior and risk patterns.

---

## 🎯 Objectives

* Analyze loan approval and rejection trends
* Identify high-risk customers
* Understand the impact of income, credit history, and demographics
* Build an interactive dashboard for insights

---

## 🛠️ Tools & Technologies

* Excel – Data cleaning and preprocessing
* SQL – Data analysis and querying
* Power BI – Dashboard and visualization

---

## 🔧 Data Preparation

* Handled missing values using mean and mode techniques
* Standardized categorical variables
* Created new features:

  * Total Income
  * Income Group (Low, Medium, High)
  * Risk Category

---

## 📊 Dashboard Features

* KPI Metrics:

  * Total Applications
  * Approved Loans
  * Rejected Loans
  * Approval Rate

* Analysis Views:

  * Credit History vs Loan Status
  * Income Group vs Approval
  * Customer Segmentation (Education, Employment, Income)

---

## 🔍 Key Insights

* Customers with no credit history have higher rejection rates
* Higher income groups show better approval probability
* Credit history is a major factor in loan approval
* Urban customers tend to have higher approval rates
